# Instructions – Génération d’un site web à partir de mon Vault

## 1. Organise le contenu
- Place chaque rubrique/section du site dans un dossier du projet.
- Crée 1 fichier .md par page du futur site, dans la rubrique appropriée.
- Ajoute images dans static/images/, CSS dans static/css/

## 2. Compile en HTML
- Utilise Pandoc pour convertir tes fichiers Markdown en HTML.
- Personnalise le rendu avec un template HTML minimal/fait-maison (optionnel).
- Copie les fichiers HTML finaux et assets statiques dans un dossier /site pour publication.

## 3. Publie si tu veux (optionnel)
- Tu peux ouvrir les HTML localement, les mettre en ligne via Netlify, GitHub Pages, ou par simple FTP.
- Tout le contenu reste versionné et synchronisé grâce à Git et Syncthing.
